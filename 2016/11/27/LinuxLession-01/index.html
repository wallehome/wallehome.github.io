Linux Lession 01 - Linux 是什么?

发表于 2016-11-27 | 分类于 Linux

1. Linux 是什么？

  • Linux是一种自由和开放源代码的类UNIX操作系统
  • 该操作系统的内核由 Linus Torvalds 在1991年10月5日首次发布
  • Linux也是自由软件和开放源代码软件发展中最著名的例子
  • 遵循GNU通用公共许可证

2. Linux 基本特性?

  • 一切皆文件
  • 每个软件都有确定的用途
  • 完全免费
  • 完全兼容POSIX1.0标准
  • 多用户、多任务
  • 支持多种平台
  • 等等…

3. Linux 文件系统

  • 普通文件 - [f] : 纯文本文件（ASCII）、二进制文件（binary）、数据格式文件(data)
  • 目录文件 - [d] -（directory）
  • 连接文件 - [l] -（link）
  • 设备与设备文件 - [b] -（device）
  • 块设备文件 - [b]
  • 字符设备文件 - [c]
  • 套接字 - [s] - （socket）
  • 管道 - [p] - （FIFO/pipe）

4. Linux 文件结构

  • “/“ : 根目录, 所有的目录、文件、设备都在”/“之下
  • “/boot” : Linux的内核及引导系统程序所需要的文件目录
  • “/home” : 存放用户的主目录
  • “/sbin” : 存放系统管理员的系统管理命令
  • “/proc” : 这个目录下获取系统信息, 这些信息是在内存中,由系统自己产生的
  • “/bin” : 普通用户常用命令、包含系统使用命令
  • “/dev” : 包含了所有linux系统中使用的外部设备
  • “/mnt” : 一般是用于存放挂载储存设备的挂载目录
  • “/lib” : 用来存放系统动态连接共享库
  • “/opt” : 主要存放那些可选的程序
  • “/srv” : 服务启动后, 所需访问的数据目录
  • “/usr” : 用户的很多应用程序和文件都存放在这个目录下
  • “/tmp” : 临时文件目录
  • “/var” : 存放系统日志文件、某些数据库默数据存放目录
  • “/selinux “ : 对SElinux的一些配置文件目录，SElinux可以让linux更加安全
